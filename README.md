*****  API automation using Python and Jenkins  *****
It was created based on fake APIs for testing from https://jsonplaceholder.typicode.com/
The code was developed using VSCode tool.
You can clone and open the folder on VSCode and click play button or open terminal type "python api.py" and press Enter.
The pipeline runs once a day.
I did not created the webhook yet so there is no page to see the last run, but I receive an email with status that could be Successful, Failed or Unusual (e.g aborted).
